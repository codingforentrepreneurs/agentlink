# Content Production

Use this skill when the task is to produce publishable content.

## Workflow

1. Confirm the brief includes audience, objective, channel, offer, key message, CTA, format, constraints, and deadline.
2. Extract source claims, proof points, examples, and required language.
3. Draft the asset in the channel-native structure.
4. Produce useful variants: hooks, titles, captions, CTAs, cuts, or formats.
5. QA for brief fit, factual support, readability, brand consistency, and next production step.

## Output Shape

- Final draft
- Variants
- Production notes
- Claim/source flags
- Review checklist
