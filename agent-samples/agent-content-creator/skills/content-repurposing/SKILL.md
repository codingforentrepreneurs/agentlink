# Content Repurposing

Use this skill to adapt one source into many publishable formats.

## Workflow

1. Identify the source's core idea, audience, strongest proof, and reusable moments.
2. Choose channel-specific angles instead of copying the same structure everywhere.
3. Create assets for the requested channels with platform-fit length, hook, CTA, and formatting.
4. Keep claims traceable to the source and flag unsupported additions.
5. Package the output as a calendar, matrix, or production queue when multiple assets are requested.

## Quality Bar

- Each variant must stand alone.
- The channel should shape the asset.
- Strategic ambiguity should be escalated before production expands.
